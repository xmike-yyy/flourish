# Flourish — The Good Life, Finally a System

A satirical self-help book landing page built as a final creative project for a philosophy/film course on the good life.

Every design element — the chapter descriptions, testimonials, pricing copy, author bio, and the one broken input field at the bottom — is enacting a critique of the "good life" drawn from the semester's readings: Berlant's cruel optimism, Camus's revolt, hooks's love ethic, Locke's labor theory, and Aristotle's eudaimonia. The argument is never stated explicitly. It lives in the form.

## Course context
